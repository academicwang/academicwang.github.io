第1课

数据类型与测量尺度

为什么数据类型如此重要？

在分析任何商业数据之前，第一步永远是：认清你手中的是什么类型的数据。因为数据类型直接决定了你可以使用哪些统计方法和图表。

举个例子：你可以计算"员工平均工资"，但你不能计算"品牌平均颜色"。颜色之间没有数学意义上的"平均值"——这就是数据类型在起作用。

核心原则：数据类型决定了适用的统计方法和运算。选错方法，分析结果就没有意义。

数据的两大阵营：分类数据 vs 数值数据

所有数据首先可以分成两大类：能不能做加减乘除？能做的是数值数据，不能做的是分类数据。

分类数据 (Categorical)

表示类别或属性，数值之间不能做数学运算。

  • 品牌名称：苹果、华为、小米
  • 颜色：红、绿、蓝
  • 满意度：非常满意、满意、一般

数值数据 (Numerical)

表示可量化的数值，可以做加减乘除。

  • 身高：175.5 cm
  • 温度：23.8°C
  • 销售额：¥125,000

四种测量尺度

统计学中，数据按照"信息量"从低到高分为四个层次。层次越高，能做的统计分析就越多。

尺度 特点 举例 允许的运算 适用统计方法
名义 (Nominal) 只有类别，无顺序 性别、品牌、血型 =, ≠ 频数、众数、卡方检验
定序 (Ordinal) 有顺序，但间距未知 满意度等级、学历 =, ≠, >, < 中位数、百分位数、秩和检验
间隔 (Interval) 有顺序，间距相等，无绝对零点 温度(°C)、年份 + , − 均值、标准差、相关系数
比率 (Ratio) 有顺序，间距相等，有绝对零点 身高、体重、销售额 +, −, ×, ÷ 所有统计方法，包括比率分析

详细解析：每种尺度的理解要点

名义尺度 (Nominal) —— "只分同类，不比大小"

名义数据是最"弱"的数据。它只能告诉我们"A 和 B 是不是一类"。比如汽车品牌：丰田、本田、宝马。你不能说"本田 > 丰田"，只能说它们是不同的品牌。

即便是用数字编码（如1=苹果，2=华为，3=小米），这些数字也不能做加减运算。3-1=2 没有任何实际意义。

判断技巧：如果把两个值互换，意义完全不变 → 名义数据。例如"性别：男=0，女=1"换成"男=1，女=0"不影响分析。

定序尺度 (Ordinal) —— "能比大小，不知差多少"

定序数据有先后顺序，但相邻等级之间的差距不一定相等。例如：

  • 酒店评分：1星 < 2星 < 3星 < 4星 < 5星
  • 但"4星到5星的差距"是否等于"1星到2星的差距"？——不一定。

因此，定序数据可以算中位数，但严格来说不适合算算术平均值（尽管实际中常被误用）。

间隔尺度 (Interval) —— "差值有意义，比值无意义"

间隔数据的特点是：两个数值之间的差是有意义的。比如今天25°C，昨天20°C，今天比昨天热5度。这个"5度"是实实在在的差距。

但是，比值没有意义。不能说"25°C是5°C的5倍热"——因为从华氏度看分别是77°F和41°F，比值变成了1.87倍。问题在于：摄氏度没有绝对零点（0°C不代表"没有温度"）。

比率尺度 (Ratio) —— "无所不能"

比率数据拥有绝对零点（0意味着"完全没有"），所以比值也有意义。比如销售额为0意味着"完全没有销售"，A公司销售额100万是B公司50万的2倍——这个2倍在任何单位下都成立。

数值数据的进一步细分

数值数据（间隔 + 比率）还可以根据取值是否连续再分为两类：

类型 定义 举例 特征
连续型 (Continuous) 取值不可数，任意两个值之间还有无限多个值 身高、体重、温度、时间 理论上可以无限精确测量
离散型 (Discrete) 取值可数的整数值 员工人数、订单数、客户投诉次数 计数得来，相邻值之间"跳变"
思考题：一个人的"年龄"算连续型还是离散型？——通常报告为整数（25岁）所以被视为离散型，但真实的年龄是连续流逝的时间，也可以视为连续型。取决于测量精度！

本节课总结框架

数据类型决策树

  1. 先做判断：能不能做加减乘除？ → 不能 = 分类数据；能 = 数值数据
  2. 分类数据再问：有没有顺序？ → 没有 = 名义；有 = 定序
  3. 数值数据再问：有没有绝对零点？ → 没有 = 间隔；有 = 比率
  4. 数值数据最后问：取值是连续还是跳跃？ → 连续 = 连续型；跳跃 = 离散型

第2课

抽样方法

为什么要抽样？

在商业分析中，我们几乎不可能调查所有对象（这叫做普查 Census）。成本太高、时间太长、甚至不可能实现。比如：

  • 想知道全国消费者的品牌偏好 → 不可能问14亿人
  • 质检部门检验产品合格率 → 不可能把每个产品都拆开检验
  • 市场调研公司估算市场规模 → 只能抽样调查

所以，抽样就是从总体中选取一部分个体进行调查，然后用这部分数据推断总体特征。抽样的核心问题是：如何保证样本能够代表总体？

核心概念：
总体 (Population)：研究对象的全体
样本 (Sample)：从总体中抽取的一部分
抽样框 (Sampling Frame)：包含所有总体成员的名单

两大类别：概率抽样 vs 非概率抽样

抽样方法首先按照"每个个体被选中的概率是否已知且非零"分为两大类：

概率抽样 (Probability Sampling) 非概率抽样 (Non-probability Sampling)
核心特征 每个个体有已知的、非零的被选概率 个体被选概率未知或为零
优点 可以计算抽样误差，结果可推断总体 操作简单、成本低、速度快
缺点 需要完整的抽样框，成本较高 无法计算抽样误差，结果不能可靠推断总体
方法 随机抽样、系统抽样、分层抽样、多阶段抽样、整群抽样 配额抽样、便利抽样、判断抽样、滚雪球抽样

六种抽样方法详解

🎲 随机抽样 (Simple Random)

从总体中完全随机地抽取样本，每个个体被选中的概率完全相等。

做法：给每个个体编号，用随机数表或计算机随机生成号码。

概率抽样 代表性强 需要完整抽样框

📏 系统抽样 (Systematic)

按固定间隔从有序总体中抽取样本。先随机确定起点，然后每隔 $k$ 个选一个。

做法：计算间隔 $k = N/n$，随机选起点 $r$（$1 \le r \le k$），然后选取 $r, r+k, r+2k, \ldots$

概率抽样 操作简单 遇周期性数据易偏差

🥧 分层抽样 (Stratified)

先将总体按某种特征分成若干层（strata），然后在每一层内进行随机抽样。

适用：总体内部差异大，但层内差异小。如按地区、年龄段分层。

概率抽样 精度高 需要分层信息

🏘️ 整群抽样 (Cluster)

将总体分成若干群体（cluster），随机抽取若干群，被抽中的群内全部调查。

适用：群内差异大，群间差异小。如按班级、社区分群。

概率抽样 成本低 精度通常较低

🔢 多阶段抽样 (Multistage)

分多个阶段逐步缩小范围。如：先随机抽省 → 再抽市 → 再抽街道 → 最后抽居民。

适用：大规模全国性调查。兼顾代表性和可操作性。

概率抽样 适合大规模 设计复杂

📋 配额抽样 (Quota)

按总体中各层的比例设定配额，调查员按配额主观选择符合条件的个体。

本质：类似分层，但不随机选，而是凭方便或判断选取。

非概率抽样 快速简便 代表性不可靠

系统抽样的数学原理

假设总体有 $N = 1000$ 人，要抽取 $n = 50$ 人的样本。

$$k = \frac{N}{n} = \frac{1000}{50} = 20$$

首先，在 $1$ 到 $20$ 之间随机选一个起点 $r$。假设 $r = 7$。那么被选中的样本编号就是：

$$7, \quad 27, \quad 47, \quad 67, \quad \ldots, \quad 987$$

也就是：$r, r+k, r+2k, \ldots, r+(n-1)k$

注意：如果总体中的个体排列存在周期性规律（如每20人一组、每组内部结构相同），且间隔 $k$ 恰好等于周期长度，系统抽样会产生严重偏差。例如工厂产品每20个一箱，抽样间隔也是20，就可能只抽到每箱的同一个位置。

分层抽样 vs 整群抽样的本质区别

这两种方法初学者容易混淆。关键区别在于：层内同质 vs 群内异质。

分层抽样 (Stratified) 整群抽样 (Cluster)
分组依据 按重要特征分组 按自然聚集或便利分组
层/群内特征 层内差异小，层间差异大 群内差异大，群间差异小
抽样方式 每层内随机抽取一部分 随机抽取若干群，群内全部调查
精度 通常精度高 通常精度较低
成本 较高（需要覆盖所有层） 较低（只需去少数几个群）
例子 按收入分层：高/中/低收入层各抽一部分 按学校分群：随机抽10所学校，校内全部学生调查

抽样方法选择决策树

有没有完整的抽样框？ → 没有 → 考虑配额/便利抽样（但结果可信度有限）
总体是否分散在地理上？ → 是 → 考虑整群或多阶段抽样
总体内部是否存在明显分层特征？ → 是 → 考虑分层抽样（精度最高）
总体规模很大但排列无周期性？ → 是 → 考虑系统抽样（操作简便）
总体规模不大、有完整名单？ → 是 → 简单随机抽样（最纯粹）

本节课小结

核心要点

  • 概率抽样：结果可以推断总体，但需要完整的抽样框和较高成本
  • 非概率抽样：操作简便快速，但无法量化抽样误差
  • 分层抽样精度高，整群抽样成本低——二者是精度与成本的权衡
  • 系统抽样要小心总体中的周期性排列
  • 配额抽样虽然方便，但调查结果不能可靠推广到总体

NumPy 数组基础

NumPy（Numerical Python）是 Python 科学计算的基础库，提供了高性能的多维数组对象。

# 导入 NumPy 库
import numpy as np

# 1. 创建一维数组（类似列表）
a = np.array([1, 2, 3, 4, 5])
print(a)          # [1 2 3 4 5]
print(type(a))    # <class 'numpy.ndarray'>

# 2. 创建二维数组（矩阵）
b = np.array([[1, 2, 3], [4, 5, 6]])
print(b.shape)    # (2, 3) — 2行3列

# 3. 常用创建函数
zeros = np.zeros((3, 4))       # 3行4列的0矩阵
ones = np.ones((2, 2))         # 2行2列的1矩阵
arange = np.arange(0, 10, 2)   # [0, 2, 4, 6, 8] — 步长为2
linspace = np.linspace(0, 1, 5) # [0.  0.25 0.5  0.75 1. ] — 均分5个数

用 NumPy 实现抽样方法

NumPy 的 random 模块提供了多种随机抽样函数。

import numpy as np

# ========== 1. 简单随机抽样 ==========
# 从 1000 人中随机抽 50 人
population = np.arange(1, 1001)  # 1~1000编号
sample_random = np.random.choice(population, size=50, replace=False)
print("随机抽样结果（前10个）:", sample_random[:10])

# ========== 2. 系统抽样 ==========
N, n = 1000, 50
k = N // n                    # 间隔 = 20
r = np.random.randint(1, k+1) # 随机起点
sample_systematic = np.arange(r, N+1, k)
print(f"系统抽样：起点={r}, 间隔={k}")
print("系统抽样结果（前10个）:", sample_systematic[:10])

# ========== 3. 分层抽样模拟 ==========
# 假设总体分三层：东部500人、中部300人、西部200人
strata = {
    '东部': (1, 500),
    '中部': (501, 800),
    '西部': (801, 1000)
}
# 按比例抽取50人：东部25、中部15、西部10
sample_sizes = {'东部': 25, '中部': 15, '西部': 10}

sample_stratified = []
for name, (start, end) in strata.items():
    layer = np.arange(start, end+1)
    s = np.random.choice(layer, size=sample_sizes[name], replace=False)
    sample_stratified.extend(s)

sample_stratified = np.array(sample_stratified)
print("分层抽样样本量:", len(sample_stratified))

Pandas 数据类型处理

Pandas 的 DataFrame 是处理表格数据的利器。它能自动识别数据类型，也可以手动指定。

import pandas as pd

# 创建模拟数据：员工信息表
data = {
    '姓名': ['张三', '李四', '王五', '赵六', '孙七'],
    '部门': ['销售', '技术', '销售', '人事', '技术'],    # 名义数据
    '职级': ['初级', '中级', '高级', '初级', '中级'],    # 定序数据
    '工龄': [2, 5, 8, 1, 4],                          # 比率-离散
    '月薪': [8500.0, 12000.0, 18000.0, 6000.0, 11000.0], # 比率-连续
    '满意度': ['满意', '非常满意', '一般', '满意', '非常满意'] # 定序
}

df = pd.DataFrame(data)
print(df)
print("\n数据类型：")
print(df.dtypes)

# 手动指定分类数据类型
df['部门'] = df['部门'].astype('category')
df['职级'] = pd.Categorical(df['职级'], categories=['初级','中级','高级'], ordered=True)
df['满意度'] = pd.Categorical(df['满意度'], 
                            categories=['非常不满意','不满意','一般','满意','非常满意'],
                            ordered=True)

print("\n修改后的数据类型：")
print(df.dtypes)

Matplotlib 基础可视化

不同类型的数据需要用不同的图表展示。

import matplotlib.pyplot as plt
import numpy as np

# 设置中文字体（根据系统可能需要调整）
plt.rcParams['font.sans-serif'] = ['SimHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

# ========== 1. 分类数据：柱状图 ==========
categories = ['销售', '技术', '人事', '财务']
counts = [35, 28, 12, 15]

plt.figure(figsize=(8, 5))
plt.bar(categories, counts, color=['#C49A6C', '#8B6F4E', '#E8D5C4', '#F5F0E8'])
plt.title('各部门人数分布（分类数据）', fontsize=14)
plt.ylabel('人数')
plt.show()

# ========== 2. 数值数据：直方图 ==========
# 模拟200名员工的月薪（正态分布）
np.random.seed(42)
salaries = np.random.normal(10000, 2500, 200)

plt.figure(figsize=(8, 5))
plt.hist(salaries, bins=20, color='#C49A6C', edgecolor='white', alpha=0.8)
plt.title('员工月薪分布（数值-连续型）', fontsize=14)
plt.xlabel('月薪（元）')
plt.ylabel('频数')
plt.show()

# ========== 3. 定序数据：条形图 ==========
satisfaction_levels = ['非常不满意', '不满意', '一般', '满意', '非常满意']
satisfaction_counts = [5, 12, 25, 48, 30]

plt.figure(figsize=(9, 5))
colors = ['#8B0000', '#CD5C5C', '#E8D5C4', '#8B6F4E', '#C49A6C']
plt.barh(satisfaction_levels, satisfaction_counts, color=colors)
plt.title('客户满意度分布（定序数据）', fontsize=14)
plt.xlabel('人数')
plt.show()
图表选择速查：
  • 名义数据 → 饼图、柱状图（只比多少，不比顺序）
  • 定序数据 → 条形图（顺序很重要，间距不重要）
  • 离散数值 → 柱状图（每个值单独计数）
  • 连续数值 → 直方图、密度图（展示分布形状）

核心要点

  • NumPy 数组是 Python 数值计算的基础，理解 shape、dtype 很重要
  • np.random.choice() 配合 replace=False 可实现不放回随机抽样
  • Pandas 的 Categorical 类型可以显式指定顺序，对定序数据处理很重要
  • Matplotlib 绘图时要选择适合数据类型的图表——数据类型和图表类型必须匹配
