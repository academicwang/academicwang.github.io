第1节课 线性函数结构

学习目标

理解线性函数的基本结构，掌握斜率和截距的经济含义，能够根据两个成本数据点求出成本方程。

核心概念：从成本场景引入线性函数

假设你经营一家小型工厂，每月的总成本由两部分组成：

  • 固定成本：无论生产多少，都要支付的租金、管理人员工资等，比如每月 ¥10,000
  • 变动成本：每生产一件产品需要原材料和人工，比如每件 ¥50

如果本月生产了 x 件产品，那么总成本 y 可以写成：

成本方程（线性函数）
$$ y = a + bx $$

其中：
\( a \) = 固定成本（截距）= ¥10,000
\( b \) = 单位变动成本（斜率）= ¥50/件
\( x \) = 产量（自变量）
\( y \) = 总成本（因变量）

例如，生产 200 件时：

y = 10,000 + 50 * 200 = 10,000 + 10,000 = 20,000 元

斜率 \( b \) 的经济含义

斜率 \( b \) 表示：当 \( x \) 增加 1 个单位时，\( y \) 增加多少。

在成本分析中，这就是单位变动成本——每多生产一件产品，总成本增加多少钱。

截距 \( a \) 的经济含义

截距 \( a \) 表示：当 \( x = 0 \) 时，\( y \) 的值。

在成本分析中，这就是固定成本——即使不生产任何产品，也必须支出的费用。

公式推导：从两点式推导斜率

假设我们只知道两个月的成本数据：

  • 1月：产量 \( x_1 = 100 \)，总成本 \( y_1 = 15,000 \)
  • 2月：产量 \( x_2 = 300 \)，总成本 \( y_2 = 25,000 \)
推导过程

假设成本方程为 \( y = a + bx \)，两个点都满足这个方程：

$$ y_1 = a + bx_1 ag{1} $$ $$ y_2 = a + bx_2 ag{2} $$

第1步：用方程 (2) 减方程 (1)

$$ y_2 - y_1 = (a + bx_2) - (a + bx_1) $$ $$ y_2 - y_1 = bx_2 - bx_1 $$ $$ y_2 - y_1 = b(x_2 - x_1) $$

第2步：两边同除以 \( (x_2 - x_1) \)

$$ b = \frac{y_2 - y_1}{x_2 - x_1} $$

第3步：代入任一点求 \( a \)

$$ a = y_1 - bx_1 $$
例题演示

数据：\((x_1, y_1) = (100, 15000)\)，\((x_2, y_2) = (300, 25000)\)

求斜率：

$$ b = \frac{25000 - 15000}{300 - 100} = \frac{10000}{200} = 50 $$

求截距：

$$ a = 15000 - 50 * 100 = 15000 - 5000 = 10000 $$

成本方程：\( y = 10000 + 50x \)

Python 绘制不同斜率和截距的直线

import numpy as np
import matplotlib.pyplot as plt

# 设置中文字体
plt.rcParams['font.size'] = 12

# 创建数据
x = np.linspace(0, 400, 100)

# 不同参数的线性函数
y1 = 10000 + 50 * x   # 原始成本线
y2 = 8000 + 60 * x    # 较低的固定成本，较高的变动成本
y3 = 12000 + 40 * x   # 较高的固定成本，较低的变动成本

plt.figure(figsize=(10, 6))
plt.plot(x, y1, label='a=10000, b=50', color='#C49A6C', linewidth=2)
plt.plot(x, y2, label='a=8000, b=60', color='#8B6F4E', linewidth=2)
plt.plot(x, y3, label='a=12000, b=40', color='#E8A87C', linewidth=2)
plt.xlabel('产量 x')
plt.ylabel('总成本 y')
plt.title('不同成本结构下的总成本曲线')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
本节课小结
- 线性函数 \( y = a + bx \) 完美刻画了"固定 + 变动"的成本结构
- 截距 \( a \) = 固定成本；斜率 \( b \) = 单位变动成本
- 已知两点即可用 \( b = \Delta y / \Delta x \) 求出完整方程

第2节课 高低点法

学习目标

掌握高低点法分解半变动成本的原理和计算步骤，理解其优缺点。

核心概念：半变动成本

在实际经营中，很多成本既不是纯固定的，也不是纯变动的，而是半变动成本（Semi-variable cost）。例如电费：有一个基本月租费（固定部分），再加上按用电量计费的部分（变动部分）。

半变动成本同样可以用线性函数表示：

$$ y = a + bx $$

但这里的 \( a \) 和 \( b \) 隐藏在混合数据中，需要通过方法把它们"分离"出来。

高低点法原理

高低点法的核心思想是：选取数据集中活动量最高和最低的两个点，用这两个极端点来确定成本线。

为什么只用两点？因为在最高点与最低点之间，成本变化不受中间波动的干扰，能最清晰地反映变动成本率。

公式推导

高低点法推导

设：

最高点：\((x_h, y_h)\) —— 最高活动量及其对应成本
最低点：\((x_l, y_l)\) —— 最低活动量及其对应成本

第1步：求单位变动成本 \( b \)

由于两点都在同一条成本线上：

$$ y_h = a + bx_h \quad ext{和}\quad y_l = a + bx_l $$

两式相减：

$$ y_h - y_l = b(x_h - x_l) $$

因此：

$$ b = \frac{y_h - y_l}{x_h - x_l} = \frac{\Delta y}{\Delta x} $$

第2步：求固定成本 \( a \)

将 \( b \) 代入任一点：

$$ a = y_h - b \cdot x_h \quad ext{或}\quad a = y_l - b \cdot x_l $$
例题演示

某工厂过去6个月的产量和电费数据：

月份 产量 x（件） 电费 y（元）
1月2005,800
2月2506,500
3月1805,500
4月3207,600
5月2806,900
6月1505,100

最高点：4月，\( x_h = 320, y_h = 7600 \)
最低点：6月，\( x_l = 150, y_l = 5100 \)

计算：

$$ b = \frac{7600 - 5100}{320 - 150} = \frac{2500}{170} = 14.71 \text{ 元/件} $$ $$ a = 7600 - 14.71 * 320 = 7600 - 4707 = 2893 \text{ 元} $$

电费方程：\( y = 2893 + 14.71x \)

阶梯式固定成本的处理

有些成本呈"阶梯式"跳跃：在一定产量范围内固定不变，超过阈值后跃升到一个新水平。高低点法对此不适用——需要回归分析（下周内容）或分段处理。

高低点法的优缺点

优点 缺点
✓ 计算简单，只需两个数据点
✓ 快速估算，适合初步分析		 ✗ 只用到两个极端点，忽略其他数据
✗ 极端点可能异常，导致偏差大
✗ 不考虑随机误差

Python 实现高低点法

import numpy as np

# 数据
months = ['1月', '2月', '3月', '4月', '5月', '6月']
x = np.array([200, 250, 180, 320, 280, 150])  # 产量
y = np.array([5800, 6500, 5500, 7600, 6900, 5100])  # 电费

# 高低点法
def high_low_method(x, y):
    # 找最高点和最低点的索引
    h_idx = np.argmax(x)  # 最高活动量
    l_idx = np.argmin(x)  # 最低活动量

    x_h, y_h = x[h_idx], y[h_idx]
    x_l, y_l = x[l_idx], y[l_idx]

    # 计算 b 和 a
    b = (y_h - y_l) / (x_h - x_l)
    a = y_h - b * x_h  # 也可用 y_l - b * x_l

    return a, b, x_h, y_h, x_l, y_l

a, b, xh, yh, xl, yl = high_low_method(x, y)
print(f"最高点: ({xh}, {yh})")
print(f"最低点: ({xl}, {yl})")
print(f"变动成本率 b = {b:.2f} 元/件")
print(f"固定成本 a = {a:.2f} 元")
print(f"成本方程: y = {a:.2f} + {b:.2f}x")
本节课小结
- 半变动成本 = 固定部分 + 变动部分
- 高低点法用两个极端点分离 \( a \) 和 \( b \)：\( b = \Delta y / \Delta x \)，\( a = y_h - bx_h \)
- 优点：简单快捷；缺点：只用两点，易受异常值影响

第3节课 散点图与最佳拟合线

学习目标

学会用散点图直观展示变量关系，理解最佳拟合线的概念，能用 Python 绘制包含拟合线的散点图。

核心概念：散点图

散点图（Scatter diagram）是将每一对数据 \( (x_i, y_i) \) 画成坐标系中的一个点。它是观察两个变量关系的第一个工具。

通过散点图，我们可以直观判断：

  • 两个变量是否有线性关系？
  • 关系是正相关（x增大y增大）还是负相关？
  • 关系的强弱程度如何？
  • 是否存在异常值？

最佳拟合线的直观概念

当散点大致呈线性分布时，我们可以画一条最佳拟合线（Line of best fit）来概括这种关系。

"最佳"的含义是：这条线到所有数据点的距离（误差）总体最小。在下一周，我们将用最小二乘法严格定义并计算这条线。

最佳拟合线的视觉标准

1. 线应该穿过数据的"中心"
2. 线两侧的点的数量大致相等
3. 所有点到线的垂直距离之和尽可能小

用 matplotlib 绘制散点图和拟合线

import numpy as np
import matplotlib.pyplot as plt

# 示例数据：产量 vs 总成本
x = np.array([100, 150, 200, 250, 300, 350, 400])
y = np.array([12000, 14000, 16500, 19000, 21000, 24000, 26000])

# 简单拟合线（用numpy的polyfit做1次多项式拟合）
coef = np.polyfit(x, y, 1)  # 1表示线性
a, b = coef[1], coef[0]  # polyfit返回 [b, a]
x_line = np.linspace(x.min(), x.max(), 100)
y_line = a + b * x_line

plt.figure(figsize=(10, 6))
plt.scatter(x, y, color='#C49A6C', s=80, zorder=3, label='实际数据')
plt.plot(x_line, y_line, color='#8B6F4E', linewidth=2, label=f'拟合线: y={a:.0f}+{b:.2f}x')
plt.xlabel('产量（件）')
plt.ylabel('总成本（元）')
plt.title('产量与总成本散点图')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

从散点图判断线性关系强弱

观察散点图时，可以从以下角度判断线性关系：

散点图特征 关系判断
点紧密分布在一条直线附近强线性关系
点大致沿某个方向分散中等线性关系
点随机分布，无明显方向无线性关系
点呈曲线趋势非线性关系

实际案例：广告费 vs 销售额

# 广告费 vs 销售额数据
ad_spend = np.array([5, 8, 12, 15, 18, 22, 25, 30, 35, 40])  # 千元
sales = np.array([50, 70, 90, 110, 125, 150, 165, 190, 210, 240])  # 千元

# 拟合
coef2 = np.polyfit(ad_spend, sales, 1)
a2, b2 = coef2[1], coef2[0]
x2_line = np.linspace(ad_spend.min(), ad_spend.max(), 100)
y2_line = a2 + b2 * x2_line

plt.figure(figsize=(10, 6))
plt.scatter(ad_spend, sales, color='#C49A6C', s=80, zorder=3, label='月度数据')
plt.plot(x2_line, y2_line, color='#8B6F4E', linewidth=2, label=f'拟合线')
plt.xlabel('广告费（千元）')
plt.ylabel('销售额（千元）')
plt.title('广告费与销售额关系')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# 预测：如果投入50千元广告费
predicted = a2 + b2 * 50
print(f"预测销售额: {predicted:.1f} 千元")
本节课小结
- 散点图是观察变量关系的可视化第一工具
- 最佳拟合线概括了数据的线性趋势
- 从点的分布紧密程度可初步判断关系强弱
