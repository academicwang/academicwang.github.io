第1节课 最小二乘法原理与回归系数推导

学习目标

完整理解最小二乘法的推导过程，能够手动计算回归系数，了解正规方程的来龙去脉。

回归分析的目的

我们有一组数据 \((x_1, y_1), (x_2, y_2), ..., (x_n, y_n)\)，想找到一条直线：

$$ \hat{y} = a + bx $$

使得这条直线能最好地"代表"所有数据点，从而可以用 \( x \) 来预测 \( y \)。

模型设定

真实的 \( y \) 不仅受 \( x \) 影响，还受其他随机因素干扰：

$$ y_i = a + bx_i + \varepsilon_i $$

其中 \(\varepsilon_i\) 是误差项（随机干扰），代表模型无法解释的部分。

最小二乘法的直观思想

对于每个数据点，预测值 \(\hat{y}_i = a + bx_i\) 和实际值 \(y_i\) 之间存在差距，称为残差（Residual）：

$$ e_i = y_i - \hat{y}_i = y_i - a - bx_i $$

最小二乘法的目标：让所有这些残差的平方和最小。为什么用平方？因为平方可以消除正负号的影响，同时放大较大误差。

残差平方和（SSE - Sum of Squared Errors）
$$ SSE = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 = \sum_{i=1}^{n} (y_i - a - bx_i)^2 $$

核心推导：从正规方程到最终公式（一步都不能跳！）

要最小化 \( SSE \)，我们对 \( a \) 和 \( b \) 分别求偏导数，并令它们等于零。

第1步：对 \( a \) 求偏导
$$ SSE = \sum_{i=1}^{n} (y_i - a - bx_i)^2 $$

把 \( SSE \) 看作 \( a \) 的函数，对 \( a \) 求导：

$$ \frac{\partial SSE}{\partial a} = \sum_{i=1}^{n} 2(y_i - a - bx_i) \cdot (-1) = -2\sum_{i=1}^{n}(y_i - a - bx_i) $$

令偏导等于零：

$$ -2\sum_{i=1}^{n}(y_i - a - bx_i) = 0 $$ $$ \sum_{i=1}^{n}(y_i - a - bx_i) = 0 $$

展开求和：

$$ \sum_{i=1}^{n} y_i - \sum_{i=1}^{n} a - \sum_{i=1}^{n} bx_i = 0 $$

注意 \( a \) 是常数，所以 \(\sum a = na\)；\( b \) 也是常数：

$$ \sum_{i=1}^{n} y_i - na - b\sum_{i=1}^{n} x_i = 0 $$

整理得到正规方程1：

$$ na + b\sum x_i = \sum y_i \tag{1} $$
第2步：对 \( b \) 求偏导
$$ \frac{\partial SSE}{\partial b} = \sum_{i=1}^{n} 2(y_i - a - bx_i) \cdot (-x_i) = -2\sum_{i=1}^{n} x_i(y_i - a - bx_i) $$

令偏导等于零：

$$ \sum_{i=1}^{n} x_i(y_i - a - bx_i) = 0 $$

展开：

$$ \sum x_i y_i - a\sum x_i - b\sum x_i^2 = 0 $$

整理得到正规方程2：

$$ a\sum x_i + b\sum x_i^2 = \sum x_i y_i \tag{2} $$
第3步：联立求解 \( a \) 和 \( b \)

从方程 (1)：

$$ a = \frac{\sum y_i - b\sum x_i}{n} = \bar{y} - b\bar{x} $$

其中 \(\bar{x} = \frac{\sum x_i}{n}\)，\(\bar{y} = \frac{\sum y_i}{n}\) 是样本均值。

将 \( a = \bar{y} - b\bar{x} \) 代入方程 (2)：

$$ (\bar{y} - b\bar{x})\sum x_i + b\sum x_i^2 = \sum x_i y_i $$

展开：

$$ \bar{y}\sum x_i - b\bar{x}\sum x_i + b\sum x_i^2 = \sum x_i y_i $$

把含 \( b \) 的项移到一边：

$$ b\left(\sum x_i^2 - \bar{x}\sum x_i\right) = \sum x_i y_i - \bar{y}\sum x_i $$

注意 \(\bar{x}\sum x_i = \frac{(\sum x_i)^2}{n}\)，\(\bar{y}\sum x_i = \frac{\sum x_i \sum y_i}{n}\)：

$$ b\left(\sum x_i^2 - \frac{(\sum x_i)^2}{n}\right) = \sum x_i y_i - \frac{\sum x_i \sum y_i}{n} $$

两边同乘 \( n \)：

$$ b\left(n\sum x_i^2 - (\sum x_i)^2\right) = n\sum x_i y_i - \sum x_i \sum y_i $$

最终得到 \( b \) 的计算公式：

$$ b = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{n\sum x_i^2 - (\sum x_i)^2} $$
第4步：等价形式（离差形式）

上面的公式可以改写为更直观的形式：

$$ b = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sum(x_i - \bar{x})^2} $$

而 \( a \) 就是：

$$ a = \bar{y} - b\bar{x} $$

验证等价性：

分子展开：\(\sum(x_i - \bar{x})(y_i - \bar{y}) = \sum x_i y_i - \bar{x}\sum y_i - \bar{y}\sum x_i + n\bar{x}\bar{y} = \sum x_i y_i - \frac{\sum x_i \sum y_i}{n}\)

分母展开：\(\sum(x_i - \bar{x})^2 = \sum x_i^2 - 2\bar{x}\sum x_i + n\bar{x}^2 = \sum x_i^2 - \frac{(\sum x_i)^2}{n}\)

所以离差形式和求和形式完全等价。

Python 手动实现最小二乘法

import numpy as np

# 数据
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 3, 5, 4, 6])
n = len(x)

# 手动计算所有求和项
sum_x = np.sum(x)
sum_y = np.sum(y)
sum_xy = np.sum(x * y)
sum_x2 = np.sum(x ** 2)

print(f"n = {n}")
print(f"sum_x = {sum_x}, sum_y = {sum_y}")
print(f"sum_xy = {sum_xy}, sum_x2 = {sum_x2}")

# 计算 b 和 a
b = (n * sum_xy - sum_x * sum_y) / (n * sum_x2 - sum_x ** 2)
a = np.mean(y) - b * np.mean(x)

print(f"\\nb = {b:.4f}")
print(f"a = {a:.4f}")
print(f"回归方程: y = {a:.4f} + {b:.4f}x")

# 用离差形式验证
x_mean = np.mean(x)
y_mean = np.mean(y)
b_check = np.sum((x - x_mean) * (y - y_mean)) / np.sum((x - x_mean) ** 2)
a_check = y_mean - b_check * x_mean
print(f"\\n离差形式验证: b = {b_check:.4f}, a = {a_check:.4f}")
本节课小结
- 最小二乘法目标：最小化 \( SSE = \sum(y_i - a - bx_i)^2 \)
- 对 \( a, b \) 求偏导 → 正规方程 → 解出 \( b = \frac{n\sum xy - \sum x \sum y}{n\sum x^2 - (\sum x)^2} \)，\( a = \bar{y} - b\bar{x} \)
- 离差形式 \( b = \frac{\sum(x-\bar{x})(y-\bar{y})}{\sum(x-\bar{x})^2} \) 与求和形式等价

第2节课 相关系数 \( r \) 与决定系数 \( R^2 \)

学习目标

理解相关系数和决定系数的含义、公式和计算方法，能够判断回归模型的拟合优度。

相关系数 \( r \) 的定义

回归方程给出了"最佳"直线，但这条直线有多"好"？我们需要一个指标来衡量 \( x \) 和 \( y \) 线性关系的强度和方向。

Pearson 相关系数
$$ r = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2 \sum(y_i - \bar{y})^2}} $$

也可以写成：

$$ r = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{\sqrt{[n\sum x_i^2 - (\sum x_i)^2][n\sum y_i^2 - (\sum y_i)^2]}} $$

\( r \) 的直观理解

\( r \) 的取值范围是 \([-1, 1]\)：

\( r \) 的值 含义
\( r = 1 \)完全正线性相关，所有点都在一条上升直线上
\( 0 < r < 1 \)正相关，\( r \) 越接近1关系越强
\( r = 0 \)无线性相关
\( -1 < r < 0 \)负相关，\( r \) 越接近-1关系越强
\( r = -1 \)完全负线性相关，所有点都在一条下降直线上

\( r \) 取值范围的直观解释

为什么 \( |r| \leq 1 \)？这可以用柯西-施瓦茨不等式证明：

令 \( u_i = x_i - \bar{x} \)，\( v_i = y_i - \bar{y} \)，则：

$$ r = \frac{\sum u_i v_i}{\sqrt{\sum u_i^2 \sum v_i^2}} $$

柯西-施瓦茨不等式告诉我们：

$$ \left(\sum u_i v_i\right)^2 \leq \left(\sum u_i^2\right)\left(\sum v_i^2\right) $$

因此 \( r^2 \leq 1 \)，即 \( |r| \leq 1 \)。等号成立当且仅当 \( u \) 和 \( v \) 成比例，即所有点恰好在一条直线上。

决定系数 \( R^2 \)

\( R^2 \)（也叫判定系数）衡量的是：回归模型能够解释多少比例的数据变异。

决定系数的三种等价定义

定义1（与 \( r \) 的关系）：

$$ R^2 = r^2 $$

定义2（变异分解）：

$$ R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST} $$

其中：

\( SST = \sum(y_i - \bar{y})^2 \) —— 总变异（Total）
\( SSR = \sum(\hat{y}_i - \bar{y})^2 \) —— 回归解释的变异（Regression）
\( SSE = \sum(y_i - \hat{y}_i)^2 \) —— 残差变异（Error）

恒等式：\( SST = SSR + SSE \)

\( R^2 \) 的直观含义

如果 \( R^2 = 0.85 \)，表示：85% 的 \( y \) 的变异可以由 \( x \) 通过线性关系解释，剩下 15% 是模型无法解释的随机因素。

Python 计算 \( r \) 和 \( R^2 \)

import numpy as np

x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 3, 5, 4, 6])

# 手动计算 r
x_mean, y_mean = np.mean(x), np.mean(y)
numerator = np.sum((x - x_mean) * (y - y_mean))
denominator = np.sqrt(np.sum((x - x_mean)**2) * np.sum((y - y_mean)**2))
r = numerator / denominator

# 手动计算 R^2
y_pred = 1.3 + 0.9 * x  # 上节课的回归方程
SST = np.sum((y - y_mean)**2)
SSR = np.sum((y_pred - y_mean)**2)
SSE = np.sum((y - y_pred)**2)
R2 = SSR / SST

print(f"r = {r:.4f}")
print(f"R^2 = {R2:.4f}")
print(f"验证: r^2 = {r**2:.4f}")
print(f"验证: SSE + SSR = {SSE + SSR:.4f}, SST = {SST:.4f}")

# 与 numpy.corrcoef 对比
r_np = np.corrcoef(x, y)[0, 1]
print(f"numpy.corrcoef: r = {r_np:.4f}")
本节课小结
- \( r \) 衡量线性关系的方向和强度，范围 \([-1, 1]\)
- \( R^2 = r^2 \) 衡量模型解释变异的比例，范围 \([0, 1]\)
- \( R^2 = SSR/SST = 1 - SSE/SST \)，恒有 \( SST = SSR + SSE \)

第3节课 Python 回归分析与预测实践

学习目标

综合运用多种 Python 工具进行回归分析，学会成本预测、销售额预测，并能通过残差分析检验模型。

方法1：numpy 手动实现（复习）

import numpy as np

x = np.array([100, 150, 200, 250, 300, 350, 400])
y = np.array([12000, 14500, 17000, 19000, 21500, 24000, 26500])

n = len(x)
sum_x, sum_y = np.sum(x), np.sum(y)
sum_xy = np.sum(x * y)
sum_x2 = np.sum(x ** 2)

b = (n * sum_xy - sum_x * sum_y) / (n * sum_x2 - sum_x ** 2)
a = np.mean(y) - b * np.mean(x)

print(f"手动计算: y = {a:.2f} + {b:.2f}x")

方法2：scipy.stats.linregress（验证）

from scipy import stats

result = stats.linregress(x, y)
print(f"scipy: a = {result.intercept:.2f}, b = {result.slope:.2f}")
print(f"scipy: r = {result.rvalue:.4f}, R^2 = {result.rvalue**2:.4f}")
print(f"scipy: p值 = {result.pvalue:.4f}")

方法3：sklearn.linear_model.LinearRegression

from sklearn.linear_model import LinearRegression

# sklearn 需要二维数组作为输入
X = x.reshape(-1, 1)
model = LinearRegression()
model.fit(X, y)

print(f"sklearn: a = {model.intercept_:.2f}, b = {model.coef_[0]:.2f}")
print(f"sklearn R^2: {model.score(X, y):.4f}")

# 预测
x_new = np.array([[450], [500]])
predictions = model.predict(x_new)
print(f"产量450时预测成本: {predictions[0]:.0f}")
print(f"产量500时预测成本: {predictions[1]:.0f}")

预测应用：成本预测

假设我们的回归方程是 \( y = 9500 + 42.5x \)，现在要预测产量为 450 和 500 时的成本：

$$ \hat{y}(450) = 9500 + 42.5 \times 450 = 9500 + 19125 = 28625 $$ $$ \hat{y}(500) = 9500 + 42.5 \times 500 = 9500 + 21250 = 30750 $$

可视化：散点图 + 回归线 + 置信区间

import numpy as np
import matplotlib.pyplot as plt

x = np.array([100, 150, 200, 250, 300, 350, 400])
y = np.array([12000, 14500, 17000, 19000, 21500, 24000, 26500])

# 回归
b, a = np.polyfit(x, y, 1)
x_line = np.linspace(50, 500, 100)
y_line = a + b * x_line

plt.figure(figsize=(10, 6))
plt.scatter(x, y, color='#C49A6C', s=80, zorder=3, label='历史数据')
plt.plot(x_line, y_line, color='#8B6F4E', linewidth=2, label=f'y={a:.0f}+{b:.2f}x')

# 标注预测点
x_pred = [450, 500]
y_pred = [a + b * xp for xp in x_pred]
plt.scatter(x_pred, y_pred, color='red', s=100, marker='*', zorder=4, label='预测点')

plt.xlabel('产量（件）')
plt.ylabel('总成本（元）')
plt.title('成本回归分析与预测')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

残差分析：检验模型合理性

残差 \( e_i = y_i - \hat{y}_i \) 应该随机分布在零线附近，没有明显模式。

# 残差分析
y_pred = a + b * x
residuals = y - y_pred

plt.figure(figsize=(10, 4))
plt.scatter(x, residuals, color='#C49A6C', s=60)
plt.axhline(y=0, color='#8B6F4E', linestyle='--')
plt.xlabel('产量 x')
plt.ylabel('残差')
plt.title('残差图（检验随机性）')
plt.grid(True, alpha=0.3)
plt.show()

# 残差统计
print(f"残差均值: {np.mean(residuals):.4f}（应接近0）")
print(f"残差标准差: {np.std(residuals):.2f}")
