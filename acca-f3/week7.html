1.1 成本分类：固定成本 vs 变动成本

在经营决策中，成本按与产量的关系分为两类：

类型定义例子特性
固定成本 (Fixed Cost, F)不随产量变化而变化的成本厂房租金、管理人员工资、设备折旧总固定成本不变，单位固定成本随产量增加而下降
变动成本 (Variable Cost, V)随产量成正比例变化的成本原材料、直接人工、计件工资单位变动成本不变，总变动成本随产量增加而上升
总成本方程：
设产量为 \( Q \)，单位变动成本为 \( v \)，总固定成本为 \( F \)
$$\text{总成本 } TC = F + v \cdot Q$$
1.2 贡献毛益（Contribution Margin）

每卖出一单位产品，收入扣除变动成本后"贡献"给覆盖固定成本的部分，叫做贡献毛益。

贡献毛益公式：
设单价为 \( P \)，单位变动成本为 \( v \) $$\text{单位贡献毛益 } CM = P - v$$
贡献毛益率（C/S ratio）： $$\text{C/S ratio} = \frac{CM}{P} = \frac{P - v}{P} = 1 - \frac{v}{P}$$

贡献毛益率告诉我们：每1元销售收入中，有多少比例可以用来覆盖固定成本和创造利润。

1.3 盈亏平衡点的推导

盈亏平衡点（Break-even Point, BEP）是利润为零时的销售量——此时总收入恰好等于总成本。

公式推导：从利润方程到盈亏平衡
第1步：写出利润方程

利润 = 总收入 − 总成本

$$\pi = P \cdot Q - (F + v \cdot Q)$$

总收入 = 单价 × 销量；总成本 = 固定成本 + 变动成本总额

第2步：整理利润方程

将含 \( Q \) 的项合并：

$$\pi = P \cdot Q - v \cdot Q - F = (P - v) \cdot Q - F$$

注意到 \( (P-v) \) 正是单位贡献毛益 \( CM \)

第3步：令利润为0，求解Q

盈亏平衡时，\( \pi = 0 \)：

$$(P - v) \cdot Q_{BE} - F = 0$$ $$(P - v) \cdot Q_{BE} = F$$ $$\boxed{Q_{BE} = \frac{F}{P - v} = \frac{F}{CM}}$$
第4步：推导金额形式的盈亏平衡点

盈亏平衡销售额 = 盈亏平衡数量 × 单价：

$$S_{BE} = Q_{BE} \cdot P = \frac{F}{P - v} \cdot P = \frac{F}{(P-v)/P}$$ $$\boxed{S_{BE} = \frac{F}{\text{C/S ratio}}$$
1.4 边际利润图（Profit-Volume Chart）

边际利润图直观展示了利润随销量变化的线性关系：

  • 横轴：销售量 \( Q \)
  • 纵轴：利润 \( \pi \)
  • 直线起点：\( (0, -F) \) —— 销量为0时，亏损等于全部固定成本
  • 直线斜率：\( CM \) —— 每多卖一单位，利润增加 \( CM \)
  • 与横轴交点：\( (Q_{BE}, 0) \) —— 盈亏平衡点

这个图清楚地告诉我们：只要知道固定成本和单位贡献毛益，就能画出整条利润线，一眼看出任意销量下的利润水平。

1.5 例题演示
例题：咖啡馆盈亏平衡分析

某咖啡馆销售一款特调咖啡，数据如下：

  • 单价 \( P = 35 \) 元/杯
  • 单位变动成本 \( v = 15 \) 元/杯（原材料+包装）
  • 每月固定成本 \( F = 8,000 \) 元（租金+工资）

要求：计算盈亏平衡销量和盈亏平衡销售额。

解答：
单位贡献毛益 \( CM = 35 - 15 = 20 \) 元/杯

盈亏平衡销量： $$Q_{BE} = \frac{F}{CM} = \frac{8,000}{20} = 400 \text{ 杯}$$
贡献毛益率： $$\text{C/S ratio} = \frac{20}{35} = 57.14\%$$
盈亏平衡销售额： $$S_{BE} = \frac{8,000}{0.5714} = 14,000 \text{ 元}$$
结论：每月卖出400杯（或销售额14,000元）即可保本。
1.6 Python实现与可视化
# CVP基础计算与可视化
import numpy as np
import matplotlib.pyplot as plt

# ===== 步骤1：定义参数 =====
P = 35          # 单价（元/杯）
v = 15          # 单位变动成本（元/杯）
F = 8000        # 固定成本（元/月）

# ===== 步骤2：计算关键指标 =====
CM = P - v                          # 单位贡献毛益
CS_ratio = CM / P                   # 贡献毛益率
Q_BE = F / CM                       # 盈亏平衡销量
S_BE = F / CS_ratio                 # 盈亏平衡销售额

print(f"单位贡献毛益: {CM} 元/杯")
print(f"贡献毛益率: {CS_ratio:.2%}")
print(f"盈亏平衡销量: {Q_BE:.0f} 杯")
print(f"盈亏平衡销售额: {S_BE:.0f} 元")

# ===== 步骤3：绘制CVP图 =====
Q = np.linspace(0, 800, 200)        # 销量范围 0~800杯
TR = P * Q                          # 总收入线
TC = F + v * Q                      # 总成本线
profit = TR - TC                    # 利润线

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# 左图：总成本-总收入图
ax1 = axes[0]
ax1.plot(Q, TR, label='总收入 TR', color='#C49A6C', linewidth=2.5)
ax1.plot(Q, TC, label='总成本 TC', color='#8B6F4E', linewidth=2.5)
ax1.axvline(Q_BE, color='#E74C3C', linestyle='--', alpha=0.7, label=f'盈亏平衡 Q={Q_BE:.0f}')
ax1.fill_between(Q, TC, TR, where=(TR >= TC), alpha=0.15, color='green', label='盈利区')
ax1.fill_between(Q, TC, TR, where=(TR < TC), alpha=0.15, color='red', label='亏损区')
ax1.set_xlabel('销量 Q（杯）', fontsize=12)
ax1.set_ylabel('金额（元）', fontsize=12)
ax1.set_title('本量利分析图', fontsize=14, fontweight='bold')
ax1.legend(loc='upper left')
ax1.grid(True, alpha=0.3)

# 右图：边际利润图
ax2 = axes[1]
ax2.plot(Q, profit, color='#2E86AB', linewidth=2.5, label='利润')
ax2.axhline(0, color='gray', linestyle='-', alpha=0.5)
ax2.axvline(Q_BE, color='#E74C3C', linestyle='--', alpha=0.7, label=f'盈亏平衡 Q={Q_BE:.0f}')
ax2.fill_between(Q, 0, profit, where=(profit >= 0), alpha=0.15, color='green')
ax2.fill_between(Q, 0, profit, where=(profit < 0), alpha=0.15, color='red')
ax2.set_xlabel('销量 Q（杯）', fontsize=12)
ax2.set_ylabel('利润（元）', fontsize=12)
ax2.set_title('边际利润图 (P/V Chart)', fontsize=14, fontweight='bold')
ax2.legend(loc='upper left')
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
第1课小结
  • 固定成本不随产量变化，变动成本与产量成正比
  • 贡献毛益 \( CM = P - v \) 是覆盖固定成本和创造利润的核心
  • 盈亏平衡销量 \( Q_{BE} = F / CM \)；盈亏平衡销售额 \( S_{BE} = F / \text{C/S ratio} \)
  • 利润方程 \( \pi = CM \cdot Q - F \) 是一切CVP分析的起点

2.1 安全边际（Margin of Safety）

安全边际衡量的是实际销售离盈亏平衡有多远——距离越远，经营越安全。

安全边际的三种表达

设实际销量为 \( Q_{actual} \)，实际销售额为 \( S_{actual} \)

数量形式
$$\text{MS}_{\text{units}} = Q_{\text{actual}} - Q_{BE}$$
金额形式
$$\text{MS}_{\$} = S_{\text{actual}} - S_{BE} = (Q_{\text{actual}} - Q_{BE}) \times P$$
比率形式（最常用）
$$\text{MS}\% = \frac{\text{MS}_{\$}}{S_{\text{actual}}} \times 100\% = \frac{Q_{\text{actual}} - Q_{BE}}{Q_{\text{actual}}} \times 100\%$$
直观理解：安全边际比率告诉你，销量最多可以下滑多少百分比才会开始亏损。例如 MS% = 30% 意味着销量下滑不超过30%都是安全的。
2.2 实现目标利润所需销量

企业不仅要保本，还要实现目标利润。从利润方程出发：

目标销量公式推导
第1步：写出含目标利润的利润方程
$$\text{Target Profit} = CM \times Q_{\text{target}} - F$$
第2步：移项求解 Q_target
$$CM \times Q_{\text{target}} = F + \text{Target Profit}$$ $$\boxed{Q_{\text{target}} = \frac{F + \text{Target Profit}}{CM} = \frac{F + \text{Target Profit}}{P - v}}$$

对比盈亏平衡公式：分子多了"目标利润"，其余结构完全相同。保本就是目标利润为0的特殊情况。

2.3 价格/成本变动的影响分析

通过敏感性分析，我们可以回答：如果涨价10%，盈亏平衡点怎么变？如果原材料涨价，利润会下降多少？

因素变动对盈亏平衡的影响直观解释
单价 \( P \) 上升\( Q_{BE} \) 下降每单位贡献更多了，需要更少的销量就能保本
单位变动成本 \( v \) 上升\( Q_{BE} \) 上升每单位贡献减少，需要卖更多才能保本
固定成本 \( F \) 上升\( Q_{BE} \) 上升需要覆盖的"坑"变大了
单价和变动成本同比例上升\( Q_{BE} \) 取决于具体变动需具体分析对 \( CM \) 的影响
2.4 多产品CVP分析

当企业销售多种产品时，不同产品的贡献毛益率不同，需要用加权平均贡献毛益率来计算综合盈亏平衡。

多产品CVP推导

假设有两种产品 A 和 B，销售比例为 \( w_A : w_B \)（金额比例）

第1步：计算加权平均C/S ratio
$$\text{Weighted C/S ratio} = w_A \times \frac{CM_A}{P_A} + w_B \times \frac{CM_B}{P_B}$$
第2步：计算综合盈亏平衡销售额
$$S_{BE}^{\text{total}} = \frac{F}{\text{Weighted C/S ratio}}$$
第3步：按销售比例分配
$$S_{BE}^{(A)} = S_{BE}^{\text{total}} \times w_A, \quad S_{BE}^{(B)} = S_{BE}^{\text{total}} \times w_B$$
2.5 例题演示
例题：安全边际与目标利润

延续咖啡馆案例：\( P = 35 \)，\( v = 15 \)，\( F = 8,000 \)，盈亏平衡 \( Q_{BE} = 400 \) 杯。

假设该店实际月销量为600杯。

要求：

  1. 计算安全边际（数量、金额、比率）
  2. 若目标月利润为4,000元，计算目标销量
解答：

(1) 安全边际：
\( \text{MS}_{\text{units}} = 600 - 400 = 200 \) 杯
\( \text{MS}_{\$} = 200 \times 35 = 7,000 \) 元
\( \text{MS}\% = 200 / 600 = 33.33\% \)

销量最多可下滑33.33%（即降至400杯）才会亏损。

(2) 目标利润4,000元：
$$Q_{\text{target}} = \frac{8,000 + 4,000}{20} = \frac{12,000}{20} = 600 \text{ 杯}$$
要实现4,000元月利润，需要月销量达到600杯——恰好就是实际销量！
例题：多产品CVP

某文具店销售两种笔记本：

  • 精装本：单价50元，变动成本30元，销售额占比60%
  • 简装本：单价20元，变动成本12元，销售额占比40%
  • 月固定成本：10,000元

要求：计算综合盈亏平衡销售额及每种产品的保本销售额。

解答：

精装本 C/S ratio = \( (50-30)/50 = 40\% \)
简装本 C/S ratio = \( (20-12)/20 = 40\% \)

加权平均 C/S ratio = \( 0.6 \times 0.4 + 0.4 \times 0.4 = 40\% \)

综合盈亏平衡销售额 = \( 10,000 / 0.4 = 25,000 \) 元

精装本保本销售额 = \( 25,000 \times 60\% = 15,000 \) 元
简装本保本销售额 = \( 25,000 \times 40\% = 10,000 \) 元

验证：精装本贡献 = 15,000 × 40% = 6,000元；简装本贡献 = 10,000 × 40% = 4,000元；合计10,000元 = 固定成本 ✓
2.6 Python：敏感性分析
# 敏感性分析：价格、成本变动对盈亏平衡的影响
import numpy as np
import matplotlib.pyplot as plt

P, v, F = 35, 15, 8000          # 基准参数
CM = P - v
Q_BE_base = F / CM

# ===== 分析：单价变动对盈亏平衡的影响 =====
price_changes = np.linspace(-20, 20, 41)   # 单价变动 ±20%
Q_BE_prices = []
for pct in price_changes:
    P_new = P * (1 + pct/100)
    CM_new = P_new - v
    Q_BE_new = F / CM_new
    Q_BE_prices.append(Q_BE_new)

# ===== 分析：变动成本变动对盈亏平衡的影响 =====
vc_changes = np.linspace(-20, 20, 41)
Q_BE_vcs = []
for pct in vc_changes:
    v_new = v * (1 + pct/100)
    CM_new = P - v_new
    Q_BE_vcs.append(F / CM_new)

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

ax1 = axes[0]
ax1.plot(price_changes, Q_BE_prices, color='#C49A6C', linewidth=2.5)
ax1.axvline(0, color='gray', linestyle='--', alpha=0.5)
ax1.axhline(Q_BE_base, color='#E74C3C', linestyle='--', alpha=0.5, label=f'基准 Q_BE={Q_BE_base:.0f}')
ax1.set_xlabel('单价变动（%）', fontsize=12)
ax1.set_ylabel('盈亏平衡销量（杯）', fontsize=12)
ax1.set_title('单价变动对盈亏平衡的影响', fontsize=14, fontweight='bold')
ax1.legend()
ax1.grid(True, alpha=0.3)

ax2 = axes[1]
ax2.plot(vc_changes, Q_BE_vcs, color='#8B6F4E', linewidth=2.5)
ax2.axvline(0, color='gray', linestyle='--', alpha=0.5)
ax2.axhline(Q_BE_base, color='#E74C3C', linestyle='--', alpha=0.5, label=f'基准 Q_BE={Q_BE_base:.0f}')
ax2.set_xlabel('变动成本变动（%）', fontsize=12)
ax2.set_ylabel('盈亏平衡销量（杯）', fontsize=12)
ax2.set_title('变动成本变动对盈亏平衡的影响', fontsize=14, fontweight='bold')
ax2.legend()
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

# 打印关键数据点
print("=== 敏感性分析关键结果 ===")
print(f"基准盈亏平衡: {Q_BE_base:.0f} 杯")
print(f"单价下降20%时, 盈亏平衡: {Q_BE_prices[0]:.0f} 杯")
print(f"单价上升20%时, 盈亏平衡: {Q_BE_prices[-1]:.0f} 杯")
print(f"变动成本上升20%时, 盈亏平衡: {Q_BE_vcs[-1]:.0f} 杯")

  • 目标销量公式：\( Q_{target} = (F + \text{目标利润}) / CM \)

3.1 存货成本的构成

存货管理的总成本由两大部分组成，它们之间存在此消彼长的关系：

成本类型定义与订货量的关系例子
订货成本 (Ordering Cost)每次订货发生的固定费用订货量越大，订货次数越少，总订货成本越低下单手续费、运输费、验收费
持有成本 (Holding/Carrying Cost)持有存货期间发生的费用订货量越大，平均库存越高，总持有成本越高仓储费、保险费、资金占用利息、损耗
3.2 EOQ模型的建立与推导

EOQ（Economic Order Quantity，经济订货量）的目标是找到使总存货成本最小的订货量。

EOQ模型推导（核心！）
第1步：定义变量
  • \( D \) = 年需求量（单位/年）
  • \( Q \) = 每次订货量（单位）
  • \( S \) = 每次订货成本（元/次）
  • \( H \) = 单位年持有成本（元/单位/年）
第2步：建立总成本方程

年订货次数 = \( D/Q \)，平均库存 = \( Q/2 \)（假设均匀消耗）

$$\text{总成本 } TC = \underbrace{\frac{D}{Q} \cdot S}_{\text{年订货成本}} + \underbrace{\frac{Q}{2} \cdot H}_{\text{年持有成本}}$$

订货成本随Q增大而减小，持有成本随Q增大而增大

第3步：对Q求导

为找到最小值，对 \( TC \) 关于 \( Q \) 求导：

$$\frac{d(TC)}{dQ} = -\frac{D \cdot S}{Q^2} + \frac{H}{2}$$

\( \frac{D}{Q} \cdot S \) 的导数是 \( -\frac{DS}{Q^2} \)；\( \frac{Q}{2} \cdot H \) 的导数是 \( \frac{H}{2} \)

第4步：令导数为0，求解EOQ
$$-\frac{D \cdot S}{Q^2} + \frac{H}{2} = 0$$ $$\frac{D \cdot S}{Q^2} = \frac{H}{2}$$ $$Q^2 = \frac{2DS}{H}$$ $$\boxed{EOQ = Q^* = \sqrt{\frac{2DS}{H}}}$$
第5步：验证是最小值（二阶导数）
$$\frac{d^2(TC)}{dQ^2} = \frac{2DS}{Q^3} > 0 \quad (\text{因为 } D, S, Q > 0)$$

二阶导数为正，说明该临界点确实是最小值点，而非最大值

第6步：计算最小总成本

将 \( EOQ = \sqrt{2DS/H} \) 代回总成本公式：

$$TC_{\min} = \frac{D \cdot S}{\sqrt{2DS/H}} + \frac{\sqrt{2DS/H}}{2} \cdot H$$

化简后（两项相等时总成本最小）：

$$\boxed{TC_{\min} = \sqrt{2DSH}}$$
经济意义：在最优订货量EOQ处，年订货成本恰好等于年持有成本。这一性质常被用来快速验证计算是否正确。
3.3 存货水平管理指标

除了确定"每次订多少"，还要确定"何时订"以及库存的上下限：

关键存货水平公式
再订货点 (Reorder Level, RL)

当库存降到这个水平时，应该立即发出新订单。

$$RL = \text{最大日用量} \times \text{最长提前期}$$

用"最大"和"最长"是为了保守估计，避免缺货

最低库存 (Minimum Level)

正常情况下不应低于的库存水平。

$$\text{Minimum Level} = RL - (\text{平均日用量} \times \text{平均提前期})$$

即：再订货点减去正常提前期内的消耗量

最高库存 (Maximum Level)

新货到达时库存可能达到的最高水平。

$$\text{Maximum Level} = RL + EOQ - (\text{最小日用量} \times \text{最短提前期})$$

假设订货到达时库存刚好降到再订货点，然后加上新订的EOQ

3.4 例题演示
例题：书店EOQ计算

某书店销售一种畅销小说，数据如下：

  • 年需求量 \( D = 2,400 \) 本/年
  • 每次订货成本 \( S = 100 \) 元/次（含物流、验货）
  • 单位年持有成本 \( H = 6 \) 元/本/年（仓储+资金占用）

要求：计算EOQ、最优订货次数、最小总成本。

解答：

$$EOQ = \sqrt{\frac{2 \times 2,400 \times 100}{6}} = \sqrt{\frac{480,000}{6}} = \sqrt{80,000} = 283 \text{ 本}$$
最优订货次数 = \( 2,400 / 283 = 8.48 \approx 9 \) 次/年
最小总成本 = \( \sqrt{2 \times 2,400 \times 100 \times 6} = \sqrt{2,880,000} = 1,697 \) 元/年

验证：订货成本 = 8.48 × 100 = 848元；持有成本 = (283/2) × 6 = 849元。两者近似相等 ✓
例题：存货水平计算

接上例，补充以下数据：

  • 年工作日：300天
  • 平均日用量：8本/天；最大日用量：12本/天；最小日用量：6本/天
  • 平均提前期：5天；最长提前期：8天；最短提前期：3天
解答：

再订货点 \( RL = 12 \times 8 = 96 \) 本

最低库存 = \( 96 - (8 \times 5) = 96 - 40 = 56 \) 本

最高库存 = \( 96 + 283 - (6 \times 3) = 96 + 283 - 18 = 361 \) 本

管理含义：当库存降至96本时发出新订单283本；正常库存应在56~361本之间波动。
3.5 Python实现：EOQ与存货监控
# EOQ计算与存货水平监控
import numpy as np
import matplotlib.pyplot as plt

# ===== 参数设定 =====
D = 2400            # 年需求量
S = 100             # 每次订货成本
H = 6               # 单位年持有成本

# ===== EOQ计算 =====
EOQ = np.sqrt(2 * D * S / H)
order_times = D / EOQ
total_cost = np.sqrt(2 * D * S * H)

print(f"EOQ = {EOQ:.1f} 本")
print(f"年订货次数 = {order_times:.1f} 次")
print(f"最小总成本 = {total_cost:.1f} 元/年")

# ===== 验证：订货成本 ≈ 持有成本 =====
order_cost = (D / EOQ) * S
holding_cost = (EOQ / 2) * H
print(f"\n验证：订货成本={order_cost:.1f}元, 持有成本={holding_cost:.1f}元")

# ===== 绘制总成本曲线 =====
Q_range = np.linspace(50, 800, 300)
TC = (D / Q_range) * S + (Q_range / 2) * H

plt.figure(figsize=(10, 6))
plt.plot(Q_range, TC, color='#C49A6C', linewidth=2.5, label='总成本 TC')
plt.axvline(EOQ, color='#E74C3C', linestyle='--', label=f'EOQ = {EOQ:.1f}')
plt.scatter([EOQ], [total_cost], color='#E74C3C', s=100, zorder=5)
plt.xlabel('订货量 Q（本）', fontsize=12)
plt.ylabel('年总成本（元）', fontsize=12)
plt.title('EOQ总成本曲线', fontsize=14, fontweight='bold')
plt.legend()
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

# ===== 存货水平模拟（锯齿图）=====
working_days = 300
daily_demand = D / working_days  # 8本/天
lead_time = 5                    # 提前期5天
RL = daily_demand * lead_time * 1.5  # 再订货点（带安全库存）

# 模拟一个订货周期内的库存变化
cycle_days = int(EOQ / daily_demand)
days = np.arange(0, cycle_days + 20)
inventory = []
inv = EOQ  # 期初库存
for day in days:
    inventory.append(inv)
    inv -= daily_demand
    if inv <= RL and day < cycle_days:  # 到达再订货点时补货
        inv += EOQ

plt.figure(figsize=(12, 5))
plt.plot(days[:len(inventory)], inventory, color='#2E86AB', linewidth=2.5)
plt.axhline(RL, color='#E74C3C', linestyle='--', label=f'再订货点={RL:.0f}')
plt.axhline(0, color='gray', linestyle='-', alpha=0.3)
plt.xlabel('天数', fontsize=12)
plt.ylabel('库存水平（本）', fontsize=12)
plt.title('存货水平波动（锯齿图）', fontsize=14, fontweight='bold')
plt.legend()
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
第3课小结
  • 存货总成本 = 订货成本 + 持有成本，两者此消彼长
  • EOQ = \( \sqrt{2DS/H} \)，通过求导令总成本最小化得出
  • 最优时，年订货成本 = 年持有成本（重要验证）
  • 再订货点、最低库存、最高库存构成存货管理的"红绿灯"系统
