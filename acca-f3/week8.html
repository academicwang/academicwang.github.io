1.1 货币时间价值的核心思想

今天的1元钱，比未来的1元钱更值钱。

因为今天的1元可以投资，在未来产生利息。这就是货币时间价值（Time Value of Money, TVM）的核心。

基本问题：如果现在把 \( P \) 元存入银行，年利率 \( r \)，\( n \) 年后能拿到多少？
反过来：如果 \( n \) 年后想拿到 \( F \) 元，现在需要存多少？
1.2 单利与复利
单利公式推导

单利：利息只按原始本金计算。

第1步：计算每年的利息
每年利息 = \( P \times r \)（只与原始本金有关）
第2步：n年后的总金额
$$\text{总利息 } I = P \times r \times n$$ $$\boxed{A = P(1 + rn)}$$
复利公式推导（逐步展开）

复利：利息计入本金再产生利息——"利滚利"。

第1期末（第1年末）
$$A_1 = P + P \times r = P(1 + r)$$

本金加上第1年的利息

第2期末
$$A_2 = A_1 + A_1 \times r = A_1(1 + r) = P(1 + r)(1 + r) = P(1 + r)^2$$

第2年的本金是第1期末的金额，不是原始本金

第3期末
$$A_3 = A_2(1 + r) = P(1 + r)^2(1 + r) = P(1 + r)^3$$
以此类推，第n期末
$$\boxed{A_n = P(1 + r)^n}$$

这是金融数学中最重要的公式之一

单利 vs 复利对比：
本金10,000元，年利率8%，5年后：
单利：\( 10,000 \times (1 + 0.08 \times 5) = 14,000 \) 元
复利：\( 10,000 \times (1.08)^5 = 14,693 \) 元
复利多出693元——时间越长、利率越高，差距越大。
1.3 名义利率与实际利率

银行说"年利率6%，按月复利"，那么实际年收益是多少？

实际年利率（EAR）推导

设名义年利率为 \( r \)，每年复利 \( m \) 次，则每期利率为 \( r/m \)。

第1步：1年后的复利终值
按复利公式，1年后： $$A = P\left(1 + \frac{r}{m}\right)^m$$
第2步：计算实际年利率
实际年利率 EAR 满足：\( P(1 + EAR) = A \) $$\boxed{EAR = \left(1 + \frac{r}{m}\right)^m - 1}$$
例题：名义利率与实际利率

某理财产品宣称"年利率6%，按月复利"。计算实际年利率。

\( r = 6\% = 0.06 \)，\( m = 12 \)（按月复利）

$$EAR = \left(1 + \frac{0.06}{12}\right)^{12} - 1 = (1.005)^{12} - 1$$ $$EAR = 1.0617 - 1 = 6.17\%$$ 实际年利率为6.17%，高于名义利率6%。
1.4 折现：从未来回到现在

投资评估需要比较"现在投入"与"未来收回"，必须把它们放到同一时间维度。

折现公式推导

已知复利终值公式 \( A = P(1 + r)^n \)，求现在的 \( P \)（现值，Present Value）。

第1步：写出复利终值方程
$$FV = PV \times (1 + r)^n$$

FV = 未来值（Future Value），PV = 现值（Present Value）

第2步：两边除以 (1+r)^n
$$\boxed{PV = \frac{FV}{(1 + r)^n}}$$

这就是折现公式——把未来的钱"打折"回到今天

第3步：引入折现因子
定义折现因子 \( DF_n = \frac{1}{(1+r)^n} \)，则： $$PV = FV \times DF_n$$

折现因子可以理解为"未来1元在今天的价值"

1.5 年金现值推导

年金（Annuity）是一系列等额的定期现金流。普通年金在每期期末支付。

普通年金现值推导（等比数列求和）

假设每年末收到 \( PMT \) 元，共 \( n \) 年，折现率 \( r \)。

第1步：写出各期现金流现值

第1年末的PMT，现值：\( \frac{PMT}{1+r} \)

第2年末的PMT，现值：\( \frac{PMT}{(1+r)^2} \)

第3年末的PMT，现值：\( \frac{PMT}{(1+r)^3} \)

……

第n年末的PMT，现值：\( \frac{PMT}{(1+r)^n} \)

第2步：年金现值 = 各期现值之和
$$PV_{\text{annuity}} = \frac{PMT}{1+r} + \frac{PMT}{(1+r)^2} + \frac{PMT}{(1+r)^3} + \cdots + \frac{PMT}{(1+r)^n}$$
第3步：提取公因子，识别等比数列
提取 \( PMT \)： $$PV_{\text{annuity}} = PMT \times \left[ \frac{1}{1+r} + \frac{1}{(1+r)^2} + \cdots + \frac{1}{(1+r)^n} \right]$$

方括号内是等比数列，首项 \( a = \frac{1}{1+r} \)，公比 \( q = \frac{1}{1+r} \)

第4步：等比数列求和公式
等比数列前n项和：\( S_n = a \cdot \frac{1 - q^n}{1 - q} \)

代入 \( a = \frac{1}{1+r} \)，\( q = \frac{1}{1+r} \)： $$S_n = \frac{1}{1+r} \cdot \frac{1 - \frac{1}{(1+r)^n}}{1 - \frac{1}{1+r}}$$
第5步：化简分母
$$1 - \frac{1}{1+r} = \frac{(1+r) - 1}{1+r} = \frac{r}{1+r}$$ 所以： $$S_n = \frac{1}{1+r} \cdot \frac{1 - (1+r)^{-n}}{\frac{r}{1+r}} = \frac{1}{1+r} \cdot \frac{1+r}{r} \cdot \left[1 - (1+r)^{-n}\right]$$
第6步：最终年金现值公式
$$\boxed{PV_{\text{annuity}} = PMT \times \frac{1 - (1 + r)^{-n}}{r}}$$

方括号内的部分叫做"年金现值系数"（PVIFA）

1.6 例题演示
例题：复利、折现与年金

(1) 现在存入50,000元，年利率5%，按年复利，10年后有多少？

(2) 3年后需要100,000元，年利率6%，现在需要存入多少？

(3) 一项投资未来5年每年末收回20,000元，折现率8%，现值是多少？

(1) 复利终值： $$FV = 50,000 \times (1.05)^{10} = 50,000 \times 1.6289 = 81,445 \text{ 元}$$
(2) 折现现值： $$PV = \frac{100,000}{(1.06)^3} = \frac{100,000}{1.1910} = 83,962 \text{ 元}$$
(3) 年金现值： $$PV_{\text{annuity}} = 20,000 \times \frac{1 - (1.08)^{-5}}{0.08} = 20,000 \times 3.9927 = 79,854 \text{ 元}$$

投资评估技术：NPV、IRR、ROCE与回收期

学习目标

  • 掌握NPV公式的逐期折现推导
  • 理解IRR的定义，掌握试错法/线性插值近似

  • 能比较四种方法的决策规则和优缺点
2.1 现金流 vs 利润：为什么投资评估用现金流？

会计利润包含了非现金项目（如折旧），且受会计政策影响。投资评估关心的是真金白银的进出。

对比维度会计利润现金流
折旧处理作为费用扣除非现金流出，不影响现金流
时间价值不考虑通过折现体现
客观性受会计政策影响更客观，可验证
再投资假设隐含利率不确定NPV假设按折现率再投资
关键原则：投资评估使用税后现金流（Cash Flow After Tax）。折旧虽不直接影响现金流，但通过抵税间接影响——这叫做"折旧税盾"效应。
2.2 净现值（NPV）

NPV是将项目所有未来现金流折现到当前，减去初始投资后的净值。

NPV公式推导

设项目初始投资为 \( I_0 \)，第 \( t \) 年末的净现金流为 \( CF_t \)，折现率为 \( r \)。

第1步：逐期折现
第1年现金流现值：\( \frac{CF_1}{(1+r)^1} \)
第2年现金流现值：\( \frac{CF_2}{(1+r)^2} \)
第3年现金流现值：\( \frac{CF_3}{(1+r)^3} \)
……
第n年现金流现值：\( \frac{CF_n}{(1+r)^n} \)
第2步：所有现值求和，减去初始投资
$$\boxed{NPV = \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t} - I_0}$$

也可以写成：NPV = −I₀ + CF₁/(1+r) + CF₂/(1+r)² + … + CFₙ/(1+r)ⁿ

第3步：决策规则
  • \( NPV > 0 \)：项目创造价值，接受
  • \( NPV = 0 \）：项目刚好回本（按折现率），可接受可拒绝
  • \( NPV < 0 \)：项目毁灭价值，拒绝
2.3 内部收益率（IRR）

IRR是使NPV = 0的折现率，代表项目内在的收益率水平。

IRR定义与线性插值近似
第1步：IRR的定义方程
IRR 满足： $$\sum_{t=1}^{n} \frac{CF_t}{(1 + IRR)^t} - I_0 = 0$$

这是一个关于 IRR 的高次方程，通常没有解析解

第2步：试错法（Trial and Error）

选择两个试算折现率 \( r_1 \) 和 \( r_2 \)，使得：

  • \( NPV(r_1) > 0 \)（即 \( r_1 < IRR \)）
  • \( NPV(r_2) < 0 \)（即 \( r_2 > IRR \)）
第3步：线性插值近似
假设NPV与折现率近似线性关系： $$\frac{IRR - r_1}{r_2 - r_1} = \frac{NPV_1 - 0}{NPV_1 - NPV_2}$$ 整理得： $$\boxed{IRR \approx r_1 + \frac{NPV_1}{NPV_1 - NPV_2} \times (r_2 - r_1)}$$

线性插值是近似方法，\( r_1 \) 和 \( r_2 \) 越接近，结果越精确

2.4 会计收益率（ROCE/ARR）

ARR使用会计利润而非现金流，是一种简单但粗糙的方法。

ARR公式
$$\text{ARR} = \frac{\text{平均年会计利润}}{\text{平均投资额}} \times 100\%$$
平均年会计利润
$$\text{平均利润} = \frac{\text{项目寿命期内总利润}}{\text{项目寿命期（年）}}$$

利润 = 现金流 − 折旧

平均投资额
如果项目结束时资产残值为 \( S \)： $$\text{平均投资额} = \frac{I_0 + S}{2}$$

假设资产价值从初始投资线性下降到残值

2.5 回收期（Payback Period）

回收期是累计现金流等于初始投资所需的时间。它回答：多久能回本？

回收期计算
非折现回收期
找到满足 \( \sum_{t=1}^{T} CF_t \geq I_0 \) 的最小 \( T \)。

若现金流不均匀，使用插值： $$\text{Payback} = T_{\text{整年}} + \frac{\text{剩余未回收金额}}{\text{下一年现金流}}$$
折现回收期（Discounted Payback）
用折现后的现金流计算： $$\sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} \geq I_0$$

折现回收期 ≥ 非折现回收期（因为折现减少了后期现金流的价值）

2.6 四种方法对比
方法决策规则优点缺点
NPVNPV > 0 则接受考虑时间价值；直接衡量财富增加折现率选择有主观性
IRRIRR > 要求回报率 则接受直观显示收益率百分比非常规现金流可能有多个IRR；互斥项目可能误导
ARRARR > 目标回报率 则接受计算简单；使用熟悉的数据忽略时间价值；使用利润而非现金流
回收期回收期 < 目标期限 则接受简单直观；强调流动性忽略回收期后现金流；忽略时间价值（非折现版）
2.7 例题演示
例题：全面投资评估

某项目需要初始投资100,000元，预计未来4年现金流如下：

  • 第1年：30,000元
  • 第2年：40,000元
  • 第3年：35,000元
  • 第4年：25,000元

折现率 \( r = 10\% \)，项目无残值，直线法折旧。

要求：计算NPV、IRR、ARR和非折现回收期。

(1) NPV计算：
$$NPV = -100,000 + \frac{30,000}{1.1} + \frac{40,000}{1.1^2} + \frac{35,000}{1.1^3} + \frac{25,000}{1.1^4}$$ $$= -100,000 + 27,273 + 33,058 + 26,296 + 17,075 = 3,702 \text{ 元}$$ NPV > 0，项目创造价值，建议接受。

(2) IRR近似：
试算 \( r = 12\% \)：NPV ≈ −1,200
已知 \( r = 10\% \)：NPV = +3,702
$$IRR \approx 10\% + \frac{3,702}{3,702 - (-1,200)} \times (12\% - 10\%) = 10\% + 1.51\% = 11.51\%$$ IRR ≈ 11.5% > 10%，与NPV结论一致。

(3) ARR计算：
年折旧 = \( 100,000 / 4 = 25,000 \) 元
各年利润 = 现金流 − 折旧：5,000 / 15,000 / 10,000 / 0
平均利润 = \( (5,000 + 15,000 + 10,000 + 0) / 4 = 7,500 \) 元
平均投资 = \( (100,000 + 0) / 2 = 50,000 \) 元
$$ARR = \frac{7,500}{50,000} = 15\%$$
(4) 回收期：
第1年末累计：30,000；第2年末：70,000；第3年末：105,000
在第3年回收：还需要30,000 / 35,000 = 0.86年
回收期 = \( 2 + 30,000/35,000 = 2.86 \) 年
第2课小结
  • NPV是投资评估的"黄金标准"，直接衡量财富增加值
  • IRR是使NPV=0的折现率，试错法+线性插值可近似求解
  • ARR使用会计利润，简单但忽略时间价值
  • 回收期强调流动性和风险，但忽略后期现金流
  • 决策优先级：NPV > IRR > 回收期 > ARR

3.1 手动实现NPV计算
import numpy as np
import matplotlib.pyplot as plt

# ===== 手动实现NPV =====
def calculate_npv(cash_flows, discount_rate):
    """
    手动计算NPV
    cash_flows: 现金流列表，第0期为初始投资（负值）
    discount_rate: 折现率
    """
    npv = 0
    for t, cf in enumerate(cash_flows):
        npv += cf / (1 + discount_rate)**t
    return npv

# 项目现金流（单位：万元）
cf_project = [-100, 30, 40, 35, 25]  # 第0-4年
r = 0.10

npv_result = calculate_npv(cf_project, r)
print(f"项目NPV (r=10%) = {npv_result:.2f} 万元")

# ===== 验证：用numpy_financial（如可用）或numpy =====
npv_numpy = npv_result  # 手动计算已完成
print(f"手动计算NPV = {npv_numpy:.4f} 万元")
3.2 手动实现IRR试错法
# ===== 手动试错法求IRR =====
def calculate_irr_trial_error(cash_flows, r_low=0.0, r_high=1.0, tol=1e-6):
    """
    用试错法（二分法）求解IRR
    要求NPV(r_low) > 0 且 NPV(r_high) < 0
    """
    npv_low = calculate_npv(cash_flows, r_low)
    npv_high = calculate_npv(cash_flows, r_high)
    
    # 确保符号相反
    if npv_low * npv_high > 0:
        raise ValueError("需要调整r_low和r_high的范围")
    
    while abs(r_high - r_low) > tol:
        r_mid = (r_low + r_high) / 2
        npv_mid = calculate_npv(cash_flows, r_mid)
        
        if npv_mid > 0:
            r_low = r_mid
        else:
            r_high = r_mid
    
    return (r_low + r_high) / 2

irr_result = calculate_irr_trial_error(cf_project, 0.0, 0.50)
print(f"项目IRR = {irr_result:.2%}")

# ===== 线性插值法验证 =====
npv_10 = calculate_npv(cf_project, 0.10)
npv_15 = calculate_npv(cf_project, 0.15)
irr_linear = 0.10 + npv_10 / (npv_10 - npv_15) * (0.15 - 0.10)
print(f"线性插值IRR (10%-15%) = {irr_linear:.2%}")
3.3 回收期计算
# ===== 回收期计算 =====
def calculate_payback(cash_flows):
    """
    计算非折现回收期
    cash_flows: 第0期为初始投资，后续为各期现金流
    """
    initial = -cash_flows[0]  # 初始投资金额
    cumulative = 0
    for t in range(1, len(cash_flows)):
        cumulative += cash_flows[t]
        if cumulative >= initial:
            # 在本期内回收
            previous_cum = cumulative - cash_flows[t]
            fraction = (initial - previous_cum) / cash_flows[t]
            return (t - 1) + fraction
    return None  # 无法回收

def calculate_discounted_payback(cash_flows, r):
    """计算折现回收期"""
    initial = -cash_flows[0]
    cumulative = 0
    for t in range(1, len(cash_flows)):
        dcf = cash_flows[t] / (1 + r)**t
        cumulative += dcf
        if cumulative >= initial:
            previous_cum = cumulative - dcf
            fraction = (initial - previous_cum) / dcf
            return (t - 1) + fraction
    return None

pb = calculate_payback(cf_project)
dpb = calculate_discounted_payback(cf_project, 0.10)
print(f"非折现回收期: {pb:.2f} 年")
print(f"折现回收期: {dpb:.2f} 年")

3.6 投资决策建议框架
系统决策框架：
  1. 初步筛选：用回收期快速排除明显不可行的项目
  2. 核心评估：计算NPV，NPV > 0是接受项目的必要条件
  3. 收益验证：检查IRR是否高于要求回报率（资本成本）
  4. 风险分析：进行敏感性分析，评估关键变量变动的影响
  5. 综合决策：NPV优先；若NPV冲突（互斥项目），选NPV最大者
综合决策案例

假设公司要求：NPV > 0，IRR > 12%，回收期 < 3年。

根据上述多项目分析结果：

  • 项目A：NPV > 0，IRR ≈ 11.5% < 12%，不满足IRR要求
  • 项目B：NPV > 0，IRR > 12%，回收期 < 3年，全部满足
  • 项目C：NPV > 0，IRR > 12%，回收期 < 3年，全部满足

决策建议：排除项目A。若项目B和C互斥（只能选一个），选NPV更高的项目B；若独立，则两个都可以接受。
